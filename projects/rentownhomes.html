RentOwnHomesPro

Client/Course:

WEB 2890 Client-Side Portfolio

Technologies:

HTML5, CSS3, Bootstrap, and Sass.

Description:

I made this site in my Client-Side Portfolio class. I was provided with site specifications from a project on 99designs.com. I was tasked with updating the look and layout of RentOwnHomesPro.

I was provided with the existing logo for the business, but I had to find the rest of the photos on Pexels and Pixabay. We were instructed to get creative with the lists on the site, so I tried to present the various lists (i.e. faqs, why rent to own, etc.) in creative ways.

Challenges:

I didn't discover until after I'd already submitted the project that the parallax scrolling I'd used on a couple of the sections didn't work on the mobile version of the site. I ended up removing parallax scrolling for the mobile version of the site.

I also struggled with one of the tools I'd chosen to use for button hover animations. However, after inspecting the code I'd imported, I was able to successfully change the color and size of the hover effect.

Resources:

FontAwesome Icons, Google Fonts, existing RentOwnHomesPro logo, pictures from Pexels and Pixabay, Hover.css, Magic.css, Bootstrap Image Hover Effects by Mike Young.

Design Decisions:

I took the base green color from the original logo. I didn't want to change the color of the logo itself, because I didn't want to interfere with any established brand recognition. I plugged the green color from the logo into the Adobe Color website and picked out a palette I thought worked well with the original green.

For the font, I wanted something dynamic, active, and bold for the headings. I found Kanit on Google Fonts and felt it fit what I was looking for. I then chose one of the recommended pair fonts for Kanit, Poppins from Google Fonts, for smaller headings and paragraph text.

For the first set of list information, I chose to break the list items up into independent div sections and pair them with relevant Fontawesome icons. This list information tells visitors why they should consider renting to own.

For the second set of list information, I chose to use Bootstrap's tabbed list component. This list information tells visitors how the rent to own process works. I paired the information in this list with FontAwesome icons as well.

One part of the project I'm most proud of is how the property cards for the properties section turned out. I like the way the colors work together on the cards. I used FlexBox to layout the icons and descriptive text at the botton of the card. The buttons at the bottom of the cards use one of the classes from the Hover.css resource I included in the project.
