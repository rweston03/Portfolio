Salazar Restaurant

Client/Course:

WEB 1400 Web Design and Usability

Technologies:

HTML5, CSS3, jQuery, and Bootstrap.

Description:

I made this site in my Web Design and Usability class. I was provided with an archived site and tasked with updating the look and layout.

I used HTML5, CSS3, and FlexBox to create a new layout for the site. I added a navbar, hero section, and hero image. I also reformatted the menu, review, and contact sections. I used jQuery and existing menu image resources to toggle in between the brunch, dinner, drinks, and dessert menus.

Challenges:

It was a challenge to get the jQuery code I'd written to work on all browsers. When I went back to the project to prepare it for display in my portfolio, I refactored the site with Bootstrap to ensure that the viewing experience was consistent across browsers.

Resources:

Google Fonts, existing menu pictures from the archived site, pictures from Pexels and Pixabay.

Design Decisions:

The archived site had a very dated look. It actually had an MC Hammer GIF dancing across the screen and felt like it was straight out of the 80's. When I looked up photos of the restaurant, it was also rocking the Saved By the Bell 80's vibe, so I decided to run with it. I pulled the teal color off of a photo of the restaurant, and then tried to put together an 80's inspired color palette.

I chose a very unique font for the hero image that would scream "80's". But, since it's a very intense font, I didn't want to use it anywhere else. So, I chose a header font that matched the hero font, but was a little more subdued. Lastly, I chose a paragraph font that was recommended by Google Fonts as a match for the header font.
